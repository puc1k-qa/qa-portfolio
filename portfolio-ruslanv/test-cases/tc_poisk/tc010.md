# TC-010

# Поиск с использованием более максимального количества символов (валидация)

# Предусловие:
- Открыта страница каталога

## Шаги воспроизведения:
1. В поле поиска ввести более 11-ти символов

## Ожидаемый результат:
1. Появляется сообщение об ошибке (красным цветом): «Максимальное количество символов — 11»
2. Опционально: Ввод блокируется на 12-м символе
