# TC-007

# Сброс результатов поиска нажатием на кнопку «крестик»

# Предусловие:
- Выполнен поиск (Кейс TC-006)

## Шаги воспроизведения:
1. Нажать на кнопку «крестик» (очистить) в строке поиска

## Ожидаемый результат:
1. Строка поиска очищается
2. Отображается полный список книг каталога (состояние до начала поиска)
