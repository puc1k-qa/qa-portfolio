# TC-006

# Поиск книг по полному имени существующего автора

# Предусловие:
- Открыта страница каталога: http://188.120.241.222/catalog

## Шаги воспроизведения:
1. В поле поиска ввести «Имя Фамилия» автора, который есть в базе
2. Нажать на иконку «Лупа»

## Ожидаемый результат:
1. В каталоге отображаются только книги данного автора
2. Счетчик «Всего {n} книг» корректно обновляется
3. Склонение слова «книга» соответствует числу (1 книга, 2 книги, 5 книг)
