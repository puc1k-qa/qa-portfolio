# TC-009

# Поиск с использованием менее минимального количества символов (валидация)

# Предусловие:
- Открыта страница каталога

## Шаги воспроизведения:
1. В поле поиска ввести менее 3-х символов
2. Нажать «Лупа» (или фокус вне поля)

## Ожидаемый результат:
1. Поиск не выполняется
2. Появляется сообщение об ошибке (красным цветом): «Минимальное количество символов - 3»
