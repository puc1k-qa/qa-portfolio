# TC-008

# Поиск книг по автору, отсутствующему в каталоге

# Предусловие:
- Открыта страница каталога

## Шаги воспроизведения:
1. В поле поиска ввести имя несуществующего автора
2. Нажать на иконку «Лупа»

## Ожидаемый результат:
1. Отображается сообщение: «По вашему запросу ничего не нашлось. Попробуйте изменить запрос»
2. Список товаров пуст (или отображается заглушка)
3. Блок сортировки и фильтров скрыт или неактивен
