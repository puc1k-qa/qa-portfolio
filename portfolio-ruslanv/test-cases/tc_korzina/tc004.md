# TC-004

# Оплата товара с валидным промокодом и чек-боксом "Есть карта лояльности"

# Предусловие:
- Товар http://188.120.241.222/catalog/1 находится в корзине
- Перейти на страницу корзины - http://188.120.241.222/cart

## Шаги воспроизведения:
1. Ввести промокод - spring
2. Поставить галочку на чек-боксе "Есть карта лояльности"
3. Нажать на кнопку "Офорить заказ"

## Ожидаемый результат:
- Промокод применился
- Появилось сообщение "Заказ на сумму 580 руб. оформлен"
