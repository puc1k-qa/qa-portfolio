# TC-012

# Проверка обработки неверного промокода

# Предусловие:
- Пользователь находится на странице корзины (/cart)
- В корзине находится как минимум один товар
- Поле для ввода промокода отображается и активно

## Шаги воспроизведения:
1. Ввести несуществующий или некорректный промокод (например, INVALID123) в поле «Промокод»
2. Нажать кнопку «Применить»

## Ожидаемый результат:
1. Система выводит сообщение об ошибке: «Промокод не найден» (или аналогичный текст, уведомляющий о невалидности)
2. Итоговая сумма заказа остается прежней, скидка не применяется
3. Фокус (курсор) сохраняется в поле ввода промокода для удобства повторного ввода
