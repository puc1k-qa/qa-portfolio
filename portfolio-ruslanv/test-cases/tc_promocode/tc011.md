# TC-011

# Проверка применения валидного промокода

# Предусловие:
- Пользователь находится на странице корзины (/cart)
- В корзине находится как минимум один товар
- Поле для ввода промокода отображается и активно

## Шаги воспроизведения:
1. Ввести корректный промокод (например, spring) в поле «Промокод»
2. Нажать кнопку «Применить»

## Ожидаемый результат:
1. Появляется уведомление: «Промокод применён»
2. Итоговая сумма заказа автоматически пересчитывается с учётом скидки
3. Применённый промокод и размер скидки отображаются в сводке заказа (раздел скидок)
