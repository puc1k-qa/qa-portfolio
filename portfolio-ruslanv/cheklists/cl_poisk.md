# Чек-лист тестирования функционала (Поиск)

## Модуль: Поиск в каталоге
- [ ] **Позитивный поиск** — отображение книг только искомого автора.
- [ ] **Логика счетчика** — корректное число и склонение слова «книга» (1 книга, 5 книг).
- [ ] **Пагинация** — пересчет количества страниц под результаты поиска.
- [ ] **Сброс поиска** — корректная работа кнопки «крестик» и возврат к полному списку.
- [ ] **Пустой результат** — отображение заглушки «Ничего не нашлось» для несуществующих данных.
- [ ] **Блокировка фильтров** — фильтры и сортировка неактивны, если поиск пуст.
- [ ] **Минимальный порог** — валидация ошибки при вводе менее 3-х символов.
- [ ] **Максимальный порог** — валидация ошибки/блокировка при вводе более 11 символов.
