# BUG-006

# Сортировка «По алфавиту по возрастанию» не меняет порядок отображения товаров.

## Окружение:
prod 188.120.241 (Google Chrome)

## Шаги воспроизведения:
1. Перейти по ссылке http://188.120.241.222/catalog
2. Выбрать фильтр Цена от 700 до 900, нажать Применить
3. Выбрать в Сортировка По алфавиту по возрастанию

## Ожидаемый результат:
Книги отсортированы в алфавитном порядке по возрастанию

## Фактический результат:
Сортировка не работает

## Severity:
Critical

## Priority:
High
