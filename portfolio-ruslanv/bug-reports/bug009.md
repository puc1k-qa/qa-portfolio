# BUG-009

# Не работает поиск в каталоге при нажатии ENTER

## Окружение:
prod 188.120.241 (Google Chrome)


## Шаги воспроизведения:
1. Открыть сайт КнигиРум
2. Перейти на таб «Каталог»
3. В поле «Поиск по каталогу» ввести любой текст
4. Нажать кнопку ENTER на клавиатуре

## Ожидаемый результат:
Отображаются результаты поиска

## Фактический результат:
Поиск не начинается

## Severity:
Major 

## Priority:
Medium
