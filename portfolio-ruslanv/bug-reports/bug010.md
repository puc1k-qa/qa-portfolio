# BUG-010

# Некорректная ссылка «author» на странице профиля автора (404 Error)

## Заголовок: 
Ошибка 404 при нажатии на гиперссылку "author" на странице конкретного автора.


## Окружение:
prod 188.120.241 (Google Chrome)


## Шаги воспроизведения:
1. Перейти на страницу со списком всех авторов: http://188.120.241.222/authors
2. Выбрать любого автора из списка (например, Стивен Кинг)
3. Проверить адресную строку (адрес изменился на .../author/Стивен-Кинг)
4. Найти на открывшейся странице гиперссылку с текстом «author» и нажать на неё


## Ожидаемый результат:
Ссылка «author» должна либо вести обратно на общий список авторов (/authors), либо быть корректно настроенным роутом. Ссылка не должна вести на несуществующую страницу 404

## Фактический результат:
Происходит переход по адресу, который выдает внутреннюю ошибку сайта: «Ошибка 404 — такой страницы не существует»

## Severity:
High 

## Priority:
Medium
