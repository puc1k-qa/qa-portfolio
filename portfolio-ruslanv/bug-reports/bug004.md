# BUG-004

# Некорректное наименование и структура API-ответа на странице авторов

# Описание:
API возвращает избыточные данные (books) вместо целевого списка (authors)

## Окружение:
prod 188.120.241 (Google Chrome)

## Шаги воспроизведения:
1. Открыть страницу http://188.120.241.222
2. Открыть панель разработчика (F12) на вкладке Network
3. Нажать на страницу ""Авторы""
4. Выбрать в списке пришедший файл с названием books
5. Открыть вкладку Response внутри этого запроса и просмотреть структуру JSON

## Ожидаемый результат:
1. Имя запроса: authors (логическое соответствие контенту страницы).
2. Тело ответа (Response): Чистый массив объектов авторов без вложенных данных о книгах.
Пример: [{""id"": 1, ""name"": ""Стивен Кинг""}, ...]

## Фактический результат:
1. Имя запроса: books (или эндпоинт, содержащий books).
2. Тело ответа (Response): Возвращается массив объектов книг, где авторы являются лишь частью данных.
3. Проблема: Клиент получает избыточный объем информации (Overfetching), который не используется на данной странице.

## Severity:
High

## Priority:
High
